# Utils Simba

A collection of utility functions for computer vision and 3D processing.

## Installation

You can install this package using pip:

```bash
# Install in development mode
pip install -e .
```

## Features

This package includes utilities for:

- Camera handling and transformations
- Configuration management
- Data reading and processing
- Depth map processing
- 3D geometry operations
- Rendering utilities
- Image processing
- Mesh handling
- Pose estimation
- Visualization tools

## Usage

```python
# Import the entire package
import utils_simba

# Or import specific modules
from utils_simba import camera, geometry, vis
```
